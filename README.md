# 🩺 AI Health Smart Assistant

An AI-powered healthcare assistant that helps users monitor symptoms, manage health records, and receive personalized health recommendations.

## 🚀 Features

- 🤖 AI Health Chatbot
- 🩺 Symptom Checker
- 📄 Health Report Analyzer
- 🍎 Diet Planner
- 💊 Medicine Reminder
- 📊 Health Dashboard
- 🔐 Secure Login
- ☕ Spring Boot Backend
- ⚛️ React + Vite Frontend

## 🛠 Tech Stack

- React
- Vite
- Java
- Spring Boot
- HTML
- CSS
- JavaScript
- REST API

## Installation

```bash
git clone https://github.com/shebendran-ece/AI-Health-Smart-Assistant.git
```

Start the backend:

```bash
mvn spring-boot:run
```

Start the frontend:

```bash
npm install
npm run dev
```
